# Citation GIF generator (Papers, Please style)

Simple Flask app that generates a small animated GIF resembling an in-game citation/stamp.

Requirements
- Python 3.8+
- Install dependencies:

```powershell
python -m pip install -r requirements.txt
```

Run (PowerShell):

```powershell
#$env:FLASK_APP = 'app.py'; python -m flask run
python app.py
```

Open http://127.0.0.1:5000 in your browser.
